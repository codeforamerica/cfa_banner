How to banner

It's simple to add the Code for America banner to your site!

In the Corner

Adding the banner to your site is as easy as adding 1 script tag.

<script src="http://codeforamerica.github.com/cfa_banner/banner.js?type=corner" id="cfa_banner_script"></script>

From the top

If you want to add it to the top of your page, just add type=flag

<script src="http://codeforamerica.github.com/cfa_banner/banner.js?type=flag" id="cfa_banner_script"></script>

Add the script right before closing body tag for best site load times.
